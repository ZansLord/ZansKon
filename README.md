## INSTALL VIA GIT

```bash
> git clone https://github.com/XrutzMalesin/MfarBot
> cd MfarBot
> npm start
```

## INSTALL VIA ZIP :

```bash
> apt upgrade && apt update
> pkg install bash
> pkg install git
> cd /sdcard
> cd MfarBot-master
> npm start
```
